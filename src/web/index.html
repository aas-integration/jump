JUMP Ultimate Math Package

Introduction

JUMP Ultimate Math Package (JUMP) is a Java-based extensible high-precision math package. Includes support for computations based on fractions. Support is included for converting to floating point numbers and BigDecimals.

JUMP is managed using SourceForge. The SourceForge project name is jump-math.

JUMP Summary page at SourceForge

Downloads and links

Downloads are distributed as .tar.gz files. There are 2 distributions. The first contains only the source files and everything else that is necessary for generating the rest. The second contains generated class files, a JAR file and API documentation as well.
